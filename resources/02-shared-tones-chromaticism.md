# Shared Tones and Chromaticism

## Key Concepts

### Shared Tones
- A **shared tone** (common pitch) helps provide consonance in chord changes
- Select a pitch to voice across a set of chord changes and build your other voices around it
- Creates smooth, connected progressions

### Chromaticism
- **Chromatic movement** (half steps) creates a feeling of movement with immediate resolution
- Use chromatic passing tones to connect chord tones smoothly
- Combines well with shared tones for balanced progressions

## Practice Examples

**Example 1: Shared Top Voice**
- Common pitch "E" shared as the top voice in all chords
- Provides consonance while lower voices move

**Example 2: Chromatic + Shared Tones**
- Mix of chromatic half-step movement and shared tones in top voice
- Balance between stability (shared) and motion (chromatic)

**Example 3: Colorful Extensions**
- Shared tone "C" voiced on top creates colorful chord extensions
- Same note sounds different over changing harmony below

## Practice Tips

- Think about how your voices move from one chord to the next
- Experiment with shared tones and chromaticism
- Select a pitch you'd like to voice in chord changes and build voices around it
- Practice identifying shared tones and chromatic movement in progressions
