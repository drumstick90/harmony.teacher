# Bass Lines Using Chord Tones

## Core Concept

Effective bass lines use **chord tones** to create harmonic movement over chord changes. The bass defines the harmonic foundation and forward motion of a progression.

## Key Principles

### 1. Build from Chord Tones First
- Use roots, 3rds, 5ths, 7ths as primary bass notes
- Create harmonic clarity and strength
- Establishes the chord quality clearly

### 2. Chromatic Movement
- Once you have strong chord tone foundation, add **chromatic passing tones**
- Chromatic notes connect chord tones via half-steps
- Creates smooth, flowing bass lines
- Adds interest without obscuring harmony

### 3. Stepwise Motion
- Prefer stepwise (scalar or chromatic) motion when possible
- Creates smooth, singable bass lines
- Easier to follow harmonically

## Practice Progression Example

The document shows a progression:
**B♭6 → Gm7 → Cm7 → F7 → B♭6**

Demonstrated in two versions:
1. **Chord tones only** - clear harmonic structure
2. **With chromatic passing tones** - more fluid, interesting movement

## Application Techniques

### For Bass Lines:
- Practice using chord tones to create movement over changes
- Improvise bass lines using chord tones as anchors
- Gradually introduce scale tones and chromaticism
- Maintain harmonic clarity even with embellishment

### For Melodies and Voice Leading:
- **Same concept applies to top voice**
- Play chord tones on top of each chord rather than in bass
- Creates melodic coherence while maintaining harmonic structure

## Practice Steps

1. **Step 1**: Play progressions using only chord tone roots
2. **Step 2**: Vary which chord tone (root, 3rd, 5th, 7th) you use in bass
3. **Step 3**: Connect chord tones with stepwise motion (scales)
4. **Step 4**: Add chromatic passing tones between chord tones
5. **Step 5**: Combine techniques for fluid, harmonically clear bass lines

## Voicing with Bass
- Try playing the bass line in left hand
- Voice chords in right hand that include the same bass notes
- Understand how bass note affects perceived chord quality
- Practice different inversions created by bass movement
